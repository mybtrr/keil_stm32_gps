# keil_stm32_gps

项目描述： 

          本项目作为pixhawk飞控跟随功能子项目，将跟随目标的位置、速度数据经gps测量以最快10Hz频率借助数传发送给飞控
          单片机通过串口2配置gps，并接收、解析位置与速度数据，最终通过串口1以固定格式发送出去
          GPS：ublox8 （须先借助u-center模块将gps波特率设置为115200）
          MCU：stm32f103rct6
